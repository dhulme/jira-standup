# Standup App

Electron app for sprint standups with embedded browser windows.

## Installation

```bash
npm install
```

## Running

```bash
npm start
```

## Features

- Left panel (200px): Standup timer at https://dhulme.uk/standup/
- Right panel (remaining width): JIRA board at https://redwoodtech.atlassian.net/jira/software/c/projects/WFM/boards/288
